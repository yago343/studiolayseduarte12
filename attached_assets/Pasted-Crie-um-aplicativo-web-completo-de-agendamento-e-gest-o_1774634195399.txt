Crie um aplicativo web completo de agendamento e gestão para um estúdio de beleza chamado "Studio Layse Duarte".

Tecnologias:

* Frontend: React + TailwindCSS
* Backend: Node.js com Express
* Banco de dados: PostgreSQL (preferencial) ou SQLite

Objetivo:
Criar um sistema moderno, limpo, profissional e totalmente responsivo (desktop, tablet e mobile), com foco em usabilidade e gestão completa do negócio.

Funcionalidades:

1. Dashboard Inteligente:

* Receita do dia, semana e mês
* Número de atendimentos
* Ticket médio
* Gráfico de receita (diário/mensal)
* Gráfico de serviços mais vendidos
* Próximos agendamentos
* Alertas (ex: agenda vazia, poucos agendamentos)

2. Gestão de Serviços:

* Criar, editar e excluir serviços
* Campos:

  * Nome
  * Descrição
  * Preço
  * Duração

3. Agenda / Calendário:

* Visualização por dia, semana e mês
* Criar, editar e cancelar agendamentos
* Selecionar cliente, serviço, data e horário
* Bloquear horários
* Status do agendamento:

  * Confirmado
  * Cancelado
  * Concluído

4. Gestão de Horários:

* Definir horários de funcionamento
* Intervalos (ex: almoço)
* Controle de disponibilidade

5. Clientes:

* Cadastro com:

  * Nome
  * Telefone
  * Email
* Histórico de atendimentos

6. Página pública de agendamento:

* Link compartilhável
* Cliente escolhe serviço, data e horário
* Interface simples e mobile-first

7. Gestão Financeira:

Receitas:

* Registrar automaticamente ao concluir um atendimento
* Campos:

  * Serviço
  * Valor
  * Forma de pagamento (Pix, dinheiro, cartão)
  * Data

Despesas:

* Cadastro manual de despesas
* Campos:

  * Descrição
  * Valor
  * Categoria
  * Data

Relatórios:

* Filtro por período (dia, semana, mês)
* Total de receitas
* Total de despesas
* Lucro líquido
* Exportação simples (opcional)

8. Configurações:

* Nome do estúdio
* Logo
* Cor principal (tema)
* Mensagens personalizadas

9. Interface e Design:

* Design limpo, moderno e minimalista
* Estilo profissional (tipo apps de salão premium)
* Uso de TailwindCSS
* Tipografia clara
* Espaçamento consistente

Responsividade:

* Mobile-first
* Adaptável para:

  * Celular
  * Tablet
  * Desktop
* Menu lateral colapsável no mobile
* Componentes ajustáveis (cards, tabelas, calendário)

Experiência do Usuário:

* Navegação simples e rápida
* Feedback visual (loading, sucesso, erro)
* Transições suaves
* Interface intuitiva

Extras:

* Dark mode (modo escuro)
* Filtros avançados
* Sistema preparado para expansão (multiusuário futuramente)

Estrutura:

* Separar frontend e backend
* Criar API REST organizada
* Código limpo e escalável

Objetivo final:
Criar um sistema completo de agendamento e gestão financeira para estúdio de beleza, com aparência profissional e pronto para uso real.
